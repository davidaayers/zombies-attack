Launch via JNLP

When Zombies Attack! is a survival-horror roguelike game.
The goal is to get to the top of the building and get rescued
by the waiting helicopter. You can choose taller buildings for harder
games.

Version 0.1 (Beta)
When Zombies Attack! is released under the GPL.
Google code page is here.
Please enter any bugs you find there, or email zombies (at) wwflgames.com
Mac and PC executables coming soon.
